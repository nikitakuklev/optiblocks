## About
This repo contains 'mostly-standalone' pieces of apsopt that might be of interest to the community.
They can be used as is by importing the package, or feel free to just copy/paste.

## Getting started
### Prerequisites
* Python >= 3.8
* pyqt >= 5.12
* pyqt-test
* pyqtgraph
